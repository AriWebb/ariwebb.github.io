# ariwebb.com

A minimal personal site: one HTML page, one stylesheet. Hosted on [GitHub Pages](https://pages.github.com/) with the custom domain `ariwebb.com`.

## Edit the site

Update copy and links in [`index.html`](index.html). Styles live in [`styles.css`](styles.css).

## Preview locally

```bash
python3 -m http.server 4000
```

Open http://localhost:4000

## Deploy

Push to `main`. GitHub Pages serves the repo root as static files (`.nojekyll` disables Jekyll).
